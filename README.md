# Hub-Banking

Custom QBCore banking resource with an original NUI, centered layout, and transparent in-game background.

## Features
- Bank locations copied from qb-banking
- Press E to open the bank UI
- Deposit and withdraw with instant balance updates
- Original UI design with glassmorphism panels

## Requirements
- qb-core
- PolyZone

## Installation
1. Place this resource in your server resources folder.
2. Add to `server.cfg`:
   - `ensure Hub-Banking`
3. Restart the server.

## Notes
- This resource focuses on cash/bank transfers only.
- Designed for FiveM NUI (transparent background in-game).
